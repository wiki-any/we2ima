**本工具是一款桌面自动化工具，旨在帮助用户将微信收藏夹中的文章导入到 ima 知识库中，使用时需遵循本软件的“使用协议”，请勿用于任何'非个人用途'及任何'非法场景'！**

**We2Ima** 是一个自动化工具，旨在将您的微信收藏夹内容自动导入到 AI 知识库（[腾讯 ima.copilot](https://ima.qq.com/)）。它将您微信中的“稍后阅读”收藏转化为结构化的、由 AI 驱动的个人知识系统。

> **注意：** 本仓库目前处于“预热”阶段。我们正在分享项目的愿景和功能特性，核心代码将分阶段发布。

## 为什么开发？
- 你有没有将**好文章/可能有用的信息**收藏下来的习惯？尤其阅读微信公众号文章时。
- 你的[微信收藏夹]是不是一直在**吃灰**？收藏的文章从来没有真正阅读过，更别提想用的时候早已忘了在那篇文章中看到过。

现在你有解决方案了！
- 将[微信收藏夹]的内容导入腾讯 ima 知识库（[ima.copilot](https://ima.qq.com/)）作为个人知识库使用。
- 可基于“单篇文章、单个知识库、跨多个知识库、全部知识库”问答，当你记不清之前看过的某个信息在哪篇文章时，直接打开[ima 知识库]提问，让AI基于你的收藏内容帮你查找，效率Up++ 10x，告别“信息黑洞”的尴尬！

## 🌟 为什么选择 We2Ima?

微信是最大的内容生态系统之一，汇聚中文互联网最优质的文本内容；ima知识库是腾讯旗下优秀的知识库产品，但截止目前，微信收藏夹中的文章依然不能自动导入ima知识库(可手动一篇一篇的操作)，缺少一款自动化工具来处理用户的收藏夹文章导入ima知识库的工作(枯燥且重复，谁爱干谁干去，反正程序小哥是忍受不了的^_^)。
- **痛点：** 收藏到“收藏夹”的文章往往会被遗忘。手动复制粘贴到 AI 工具既繁琐又缓慢。
- **解决方案：** We2Ima 实现了整个流程的自动化，通过模拟人类行为来“阅读”并“导入”公众号文章，从而高效且安全地构建您的 AI 个人知识库。

## 🚀 核心功能

- **🤖 智能 UI 自动化**：基于 GUI自动化 开发。它不仅仅是简单的“点击”，而是模拟了人类的鼠标操作行为，以确保兼容性和安全性。
- **🧠 深度 ima 集成**：针对腾讯 ima 知识库（[ima.copilot](https://ima.qq.com/)）进行了专门优化，包括自动菜单交互和状态监测。
- **🔄 智能同步模式**：
  - **增量扫描**：仅同步自上次运行以来新增的文章。
  - **日期范围模式**：专注于特定时间段的内容。
  - **跳过已导入**：利用基于 SQLite 的历史记录管理，实现智能重复检测。
- **📅 日期标准化**：自动将微信的相对日期（“今天”、“昨天”、“3天前”）转换为绝对标准格式，实现精确追踪。
- **🛡️ 模拟人类的鼠标操作行为**：具备随机阅读行为、空闲模式和可配置的速度，模拟真实用户交互。
- **🖥️ 现代 GUI 界面**：简洁直观的界面，支持深色模式和系统托盘操作。
- **🌐 多语言支持**：内置英文和简体中文的完全本地化支持。
